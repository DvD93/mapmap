MapMap es una aplicación open source, web y móvil que te permite mapear rutas que recorren los vehiculos del Transporte Público de una ciudad, con el objetivo de generar datos en formato GTFS.

Con está aplicación puedes crear trazos georeferenciados sin la necesidad de consumir datos del dispositivo móvil, podrás capturar datos como: paradas, pasajeros, tiempo de recorridos, pasajeros que abordan y bajan en una parada, entre otras.
